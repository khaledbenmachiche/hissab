SOAP Web Service Information

HISSAB provides SOAP web services for mathematical expression evaluation and OCR processing.

Service Endpoints:

WSDL: http://localhost:8080/hissab-web/HissabService?wsdl
Service URL: http://localhost:8080/hissab-web/HissabService

Available Operations:

  • calculateFromString(String expression)
    Evaluates mathematical expressions like "2+3*4", "10-5", "(2+3)*4"
  • calculateFromImage(byte[] image)
    Simulates OCR processing and evaluates extracted expressions
  • healthCheck()
    Returns service health status

How to Use:

  • Desktop Client: Run the Swing GUI application from the hissab-client module
  • SOAP Testing: Use SoapUI or similar tools with the WSDL URL above
  • Integration: Generate client stubs using wsimport and the WSDL

Database Access:

  • phpMyAdmin: http://localhost:8081
  • Database: hissab_db
  • Table: trace (logs all calculations)
